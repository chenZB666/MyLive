# MyLive
跨平台桌面应用，支持在统一界面内观看斗鱼、虎牙、B站、抖音四个主流直播平台的直播流和实时弹幕，以及平台登录、订阅关注、开播提醒等功能。
## 技术栈

| 类别 | 选型 |
|------|------|
| 桌面框架 | Electron 33+ |
| 构建工具 | electron-vite |
| 前端框架 | React 19 + TypeScript |
| UI 组件库 | Ant Design |
| 状态管理 | Zustand |
| 样式方案 | Tailwind CSS |
| 视频播放 | flv.js + hls.js |
| 本地存储 | electron-store |
| 通知 | Electron Notification API |
